# 📝 Lista de Tarefas (To-Do List)

Uma aplicação simples, moderna e eficiente para gerenciamento de tarefas diárias. O projeto foi desenvolvido com foco em uma interface limpa para o usuário e uma estrutura de código organizada.

---

## 🚀 Tecnologias Utilizadas

O projeto combina tecnologias fundamentais de desenvolvimento web para garantir uma experiência fluida:

*   **Front-end:** HTML5, CSS3 e JavaScript (ES6+) para manipulação dinâmica de elementos e interface responsiva.

---

## ⚙️ Funcionalidades

*   ➕ **Adicionar Tarefas:** Insira novas atividades rapidamente.
*   ✅ **Concluir Tarefas:** Marque as tarefas finalizadas para manter o controle do seu progresso.
*   ❌ **Remover Tarefas:** Exclua itens da lista de forma simples.
*   📱 **Design Responsivo:** Interface adaptada para visualização em dispositivos móveis e desktop.
